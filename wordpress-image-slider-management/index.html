WordPress Image Slider Management

A lightweight, customisable and smart approach to WordPress image slider management

May 18th, 2013

For the behaviour side of things, we'll be using FlexSlider, a really cool, lightweight (16bk minified), compatible, and touch friendly, jQuery slider. However, feel free to run this with anything you choose. We are just concerned with the mark up and image management.

FlexSlider project page: http://www.woothemes.com/flexslider/

FlexSlider GitHub repo: https://github.com/woothemes/FlexSlider

We will be using WordPress Custom Post Type, and WordPress Custom Taxonomy classes to create our content management.

JavaScript

https://github.com/woothemes/FlexSlider/blob/master/jquery.flexslider-min.js

;(function($) {
  $('.flexslider').flexslider({
    animation: "slide"
  });
})(jQuery);

CSS

Build upon this CSS, but this is enough to get things looking right.

.flexslider img {
    width: 100%;
    height: auto;
}

.flexslider ul {
    margin: 0;
    padding: 0;
}

Function files

Kick things off by placing the following code where ever you like to keep the functionality and config php scripts for your theme.

<?php
/**
 * Image slider config
 */

/* Add custom image sizes */
add_action('init', 'lt3_add_image_sizes_project');
function lt3_add_image_sizes_project()
{
  add_image_size('large-hero-image', 960, 395, true);
}

/* Include the Flexslider */
add_action('wp_enqueue_scripts', 'lt3_enqueue_slider_scripts');
function lt3_enqueue_slider_scripts()
{
  wp_enqueue_script('flexslider'
    , get_stylesheet_directory_uri() . '/path/to/jquery.flexslider-min.js'
    , array('jquery')
  );
}

/* Create slider image post type */
$Slider_Image = new LT3_Custom_Post_Type('slider_image'
  , array()
  , array(
    'supports' => array('title', 'thumbnail')
  )
);

/* Create the slider region taxonomy */
$Slider_Region = new LT3_Custom_Taxonomy('slider_region'
  , 'slider_image'
);

The above code has: - Brought in the required class scripts, - Created an image size that we can use for our output, - Declared the image slider post type, and region taxonomy.

Template page

Within the template page where we are going to display a particular image slider region, place the following code.

<section class="flexslider">
<?php
global $Slider_Image, $Slider_Region;
$region = 'home-page';
$size   = 'large-hero-image';
$args   = array(
  'tax_query' => array(
    array(
      'taxonomy' => $Slider_Region->name,
      'field'    => 'slug',
      'terms'    => $region
    )
  )
);
$slide_images = $Slider_Image->get($args);
if ($slide_images) : ?>
<ul class="slides">
<?php foreach ($slide_images as $image) : if (has_post_thumbnail($image->ID)) : ?>

  <?php $attr = array('alt' => $image->post_title); ?>
  <li><?php echo get_the_post_thumbnail($image->ID, $size, $attr); ?></li>

<?php endif; endforeach; ?>
</ul>
<?php endif; ?>
</section>
